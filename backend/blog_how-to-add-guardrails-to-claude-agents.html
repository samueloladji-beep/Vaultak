Frequently Asked Questions

How do I add guardrails to a Claude agent?
Intercept tool calls with vt.check() before execution and use Vaultak policies to define what the agent can and cannot do.
Can Vaultak protect against prompt injection in Claude agents?
Yes. Vaultak behavioral deviation detection flags actions inconsistent with the agent configured purpose.
How do I audit what my Claude agent did?
Every tool call is logged by Vaultak with timestamps, risk scores, and full argument details at app.vaultak.com.
Does Vaultak work with Claude tool use API?
Yes. Vaultak works at the tool execution layer and is compatible with all Claude models that support tool use.
